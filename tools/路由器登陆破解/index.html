# 破解过程

破解 wifi 密码，见下一篇博客

根据博主学习状况不断更新进度

  1. 能在浏览器进入登陆界面

  2. 制作密码包暴力破解 (已得到最全密码包和生成密码软件)

  3. python 脚本：

  • 方式一：
    • 了解登录页面的处理逻辑 --JS 代码、html 页面
    • 读取密码包赋值给 input
    • 模拟页面点击事件提交
  • 方式二：
    • 抓包得到传送 url
    • 不断模拟浏览器向 API 发送 fake msg
    • 处理返回值
    • 如果返回非 ERROR 或者原页面的是破解成功
