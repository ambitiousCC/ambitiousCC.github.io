# 关于博客

上一个博客由于换电脑时候没有备份导致所有的内容丢失，只有之前的图片和 github 托管的内容了。拖了好久之后，我索性换了一个新的主题来托管博客。

这里是相对于 csdn 上面更全面的个人博客的内容。

建立博客的初衷不变，是我发现搜索到的很多内容并不能很好的解决个人问题，往往解决一个问题要搜索很多资料。

相对于上一任博客，这个尽可能减少花里胡哨的东西，提高博客的加载速度。

同时这个博客可以作为某些项目的发布文档，我会好好备份的。

博客的内容会涉及很多方面，主要是包括了我所学到的各种知识，尤其是遇到的各种坑。

由于博客内容的来源肯定非常复杂，我不能全部贴上所有的相关资料的整合者，所以希望原创作者能多多包涵。涉侵请电邮 derekcuiqin@gmail.com

由于个人对美少女的喜爱，我会专门开一个秘密模块作为恋爱日记。

# 关于博主

博主比较极端。懒散的时候懒散至极，学习的时候全身心都会投入，所以一个好的学习环境和博主自己的心态非常大的影响到了博主的学习生活。

随着年龄的增长，父母的退休年龄一天天的到来，博主对金钱的追求越来越大。但是这些都是博主学习的动力，没有啥除了自己是阻挡博主前进的动力的。

博主科班出身，主役大数据相关。不停的研究爬虫和数据可视化、机器学习和深度学习。学的越深觉得水越深，头越凉。

# 关于心态

在博主的第一任博客正式 deploy 的时候，博主暗自维护好自己的博客，但是最终因为自己太过懒散和主题的不易扩展性差不多丢弃了上传文章的习惯，所以这次找了功能丰富的 casper 作为主题。

发现了影响自己学习的最大的因素就是一些复杂的过程影响了自己学习的过程。就像现在能够纯粹的通过敲打键盘来书写 ABOUT 模块，而不需要调配各种参数或者去找图片。

最后上一次博客是 19 年的八月份，那时候还没有遇到这么可爱的女票，他也是我学习的动力之一。
