# 我喜欢你什么？

👉关于我的思考

谈恋爱不是一件小事，如果我们互相答应，那么就是押上了自己的一辈子哦。❤

虽然我对你早就有好感了😁，但是我并不希望我没有对你的人生、性格等进行足够的了解就贸然堂堂陈词只不过是见色起意罢了 (是不是🐷)。每一对恋人难道不都是追求着相遇后相知，相知后相随嘛。认真回想自己以往的十几年，抛开不懂事的那几年，我虽没有丰富的情感经历，但是各种小说电影足够让我对人类这一复杂的情感做出了大概的认识。什么是喜欢？什么是爱？也许我们现在仅仅是喜欢彼此，有着一定的好感，时间继续推移下去，爱就会萌发了，当然有着不幸的话爱也会消散。😭

我想，你一定会认认真真仔仔细细的问我：“我喜欢你什么？🧐”。嘿嘿嘿对不住，在我说出 “我喜欢你” 那句话之前，我就要做好充分十足的准备来回答你的问题。👍(在写这篇简单的散文时，我当然并没有说那句话，我也不清楚现在是发什么情况，不过我会把这篇文修修补补得更加舒服贴切)

首先，我们真的很像。在自己的生活中总是有着忙不完的事情，但是有时候又会突然闲下来不知道自己要干些什么，然后就开始刷手机了，追剧开始！我不是追剧，但是也差不多，悄悄说一下，我一般是看看知乎里面的各种奇葩问题，哈哈哈挺有意思的😀。咳咳，回到正题，我和你都追求着优秀，但是有时候发现自己真的什么也不会呀，没关系的，现在什么都能继续学下去！我一度最大的愿望就是能找到一个能和自己一起学习、进步、成长的人。我是一个内心非常孤独的人，从小到大，一直一直都是一个人。中学的时候放学回家，最好的小伙伴不能陪我到家，周末，一个人在家学习看书，只有在放寒暑假的时候，才可以和我最亲爱的表弟，一起玩玩游戏，上学了就删除了哈哈哈。到了大学，每天都在忙自己的事情，偶尔去吃饭与实验室同伴聊聊新点子，回宿舍与室友谈谈作业之类的 (我不玩游戏的)，嗯，我是多么希望能有一个人可以陪我度过只有我一个人的时光 -- 在我一个人回宿舍的时候、我一个人出去旅行的时候、我一个人去自习室认真刷题的时候、一个人去食堂吃饭的时候・・・(这里说的有点简略，篇幅不能太长不然看不下去了嘿嘿嘿)。也是这个原因，我渐渐的发现了自己对你的好感，是你给了我改变的动力，学习英语的动力 (感觉并没有多少😅)，(: 咦这里好像可以删掉)，让我变得越来越好。哈哈哈现在我害羞了你也看不到。

然后，十分十分感谢你能和我每天都去跑运动校园 (真的怀念呀🤭)，有你陪伴的感觉真好🎁，怎么说呢，一直处于恋人未满朋友以上的关系，我什么都想给你说，哈哈哈我是没有多少秘密的，这让我感觉很轻松。渐渐地我发现，有的话题我只想和你说了，每天打开手机，最期待的是你的昵称 + 说话的内容，看到其他人和某些群 @全体成员有种很失落的感觉。嗯，在我多次发现，“咦，她干什么去了呀？为什么没有回复我的消息呢？”“恩，她应该是在学习吧” 过一会儿” 她还是没有回复我的消息！？“” 她还在学习吧 “再过一会儿就忍不住想给你发可爱的表情包或者臭熊猫了嘿嘿嘿，当你给我回信的时候，内心可别说有多激动了。现在觉得，只有能见到你的时间才是真正的假期，现在到了周末虽然课不多了，事多了，却常常很难和你相遇。咳咳，如果这样都骗自己说我们只是好的同学，那么我自己可以准备凭实力单身几辈子吧。然后要感谢你给我推荐的这本《每一天梦想练习》，仿佛把你自己推荐给我一样，我见到一个不断成长的另维，也见到一个不断成长的你。在遇到你之前，我一直想的仅仅是快点考证，学习、做项目和拉高绩点，没有认真地思考未来自己的生活是什么样子，自从遇到你之后，我找到了更多努力的理由，比如能让你幸福的话，简直不要太美好了。

最后，不要说太多啦，我不喜欢一下子就把自己全部都倒出来🤔，希望你能一点一点地认真的认识我。(如果你能看到这篇文章🤯) 我真的很幸福呀，以前看到的故事都是一个人苦苦的追求着另一个人，比如《少年维特的烦恼》里的维特喜欢上了有男朋友的人 -- 这真的太惨了，又比如《大鱼海棠》里面的白毛男主，可怜女主一直把他当做亲兄弟。所以，能够遇到两厢情愿的喜欢 + 如此可爱优秀的你，也许是我上辈子拯救了银河系吧嘻嘻嘻，(L) lucky！很幸运你出现在我的生命中❀。

其实吧，我们两个都算是大忙人，每天有的时间并不多，可能平时在学校也不能时刻的待在一起 (事实证明确实如此🖊)，也不会一直不断的在社交软件📱上面聊天 (我觉得在网络上不能给对方呈现最真实的自己，比如说我就常常线上蹦迪，线下就变成木头人了哈哈哈哈)，我更不希望严重影响到你的学习生活，所以，就当做平常一样对待这份珍宝吧。然后，希望能在以后，彼此能够很爽朗地告诉别人你是我的人，不用着躲躲藏藏 (说的就是你😡)。我们都会有缺点，现在是对方的老师，如果有什么做的不好的地方，希望我们都能正视问题好好改正☺。

还是最后，正如你说的不将就😱，我希望我们都不要将就对方，遇到自己不舒服的事情就提出来，既然能走到一起，什么问题不能解决的呢！然后是我们虽然在一起了，也要考虑一下以前喜欢自己的人，就别发空间秀恩爱撒狗粮啦，简简单单的情头情话情侣空间一条龙了解一下 (一百天的崔崔过来吐槽了✌)？我会先把你介绍给我最好的朋友们 (生日一起过了✨。

啊，还有好多想说的堵在心里，以后想到什么就补充一下吧🙃。

你能成为我的女朋友，这里有好多想和你一起做的事情！！！！交换日记怎么样呀 (空)，做个照片框怎么样呀 (已经存了很多照片😏)，emmm 感觉是不是有点麻烦呢。哎，谈恋爱好麻烦的话，以后就要麻烦你了 (事实证明，麻烦的是我😜)。
