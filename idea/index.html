# 自动规划学习任务

临近期末了，各种考试同时出现在你的日程表上，最坑爹的是我四天考七科。

所以安排复习计划就显得非常重要，有没有这样一个程序能够智能的安排自己的学习计划呢，我们好好分析一下需求。

# 具体功能

用户输入自己的考试科目的考试时间，以及这段时间无法复习的时间，考试科目的重要程度，考试科目的难易程度。
系统自动输出规划好的详细的学习路线。

要求：不得占用日常生活的作息，保证正常的时间作息

补充功能：根据用户输入的科目掌握程度以及当前所剩下的复习时间，结合网络对此门科目的评价和难易程度，预测用户可能得到的分数。用户根据自己可能的分数调整自己的作息，冲冲冲！

# 生产空间

APP / 小程序 / 网站

# 能够实现需要的

开发者需要熟悉至少一门编程语言实现，主要实现预测功能和动态规划功能，对算法设计的要求较高。

  1. 具有良好的图形界面和用户交互能力
  2. 具有机器学习算法和足够的知识库预测用户分数
  3. 具有优化的算法尽快的输出最优的用户复习计划
  4. 具有根据用户习惯提出的学习建议
